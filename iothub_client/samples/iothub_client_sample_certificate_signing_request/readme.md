# Azure IoT Hub Client Sample: Certificate Renewal via CSR

This sample demonstrates how to renew a device certificate using Certificate Signing Requests (CSR) on IoT Hub.

The device connects with an existing X.509 certificate, sends a CSR to obtain a renewed certificate, then reconnects using the renewed certificate to verify it works.

For initial device provisioning with CSR via DPS, see the [prov_dev_client_ll_x509_csr_sample](../../../provisioning_client/samples/prov_dev_client_ll_x509_csr_sample/).

## Build instructions

1. Modify `.\iothub_client_sample_certificate_signing_request.c`: update the `connectionString`,
   `x509certificate` and `x509privatekey` variables with your device's IoT Hub connection info and
   current X.509 certificate and private key.

1. Set the variables `certificateSigningRequest` and `csrPrivateKey` as:
   - `certificateSigningRequest`: PEM-encoded certificate signing request.
   - `csrPrivateKey`: P256 EC private key in PKCS8 PEM format corresponding to the CSR.

   For **non-production test-only** purposes, these can be generated with the commands below. Copy and replace the variable definitions generated by these scripts into `.\iothub_client_sample_certificate_signing_request.c`.

   Linux:

   ```bash
   DEVICE_ID="your device id"
   CSR_KEY_PEM_FILE_PATH=$(pwd)/${DEVICE_ID}-csr-private-key.pem

   openssl ecparam -name prime256v1 -genkey -noout | openssl pkcs8 -topk8 -nocrypt -out $CSR_KEY_PEM_FILE_PATH
   CSR_PEM=$(openssl req -new -key $CSR_KEY_PEM_FILE_PATH -subj "/CN=$DEVICE_ID")

   FORMATTED_CSR_KEY_PEM=$(cat $CSR_KEY_PEM_FILE_PATH | sed "s/^/\"/g" | sed "s/$/\\\n\"/g" | sed '$s/$/;/')
   echo "static const char* csrPrivateKey ="; echo "$FORMATTED_CSR_KEY_PEM"

   FORMATTED_CSR_PEM=$(echo "$CSR_PEM" | sed "s/^/\"/g" | sed "s/$/\\\n\"/g" | sed '$s/$/;/')
   echo "static const char* certificateSigningRequest ="; echo "$FORMATTED_CSR_PEM"
   ```

   Windows (PowerShell):

   ```powershell
   $DEVICE_ID = "your device id";

   $privateKey = [System.Security.Cryptography.ECDsa]::Create([System.Security.Cryptography.ECCurve]::CreateFromFriendlyName("nistP256"))
   if ($PSVersionTable.PSVersion.Major -lt 7) {
      $base64pkcs8PrivateKey = [Convert]::ToBase64String($privateKey.Key.Export([System.Security.Cryptography.CngKeyBlobFormat]::Pkcs8PrivateBlob), 'InsertLineBreaks')
   } else {
      $base64pkcs8PrivateKey = [Convert]::ToBase64String($privateKey.ExportPkcs8PrivateKey(), 'InsertLineBreaks')
   }

   $dn = New-Object System.Security.Cryptography.X509Certificates.X500DistinguishedName("CN=$DEVICE_ID")
   $csr = New-Object System.Security.Cryptography.X509Certificates.CertificateRequest($dn, $privateKey, [System.Security.Cryptography.HashAlgorithmName]::SHA256)
   $CSR_BASE64 = [Convert]::ToBase64String($csr.CreateSigningRequest())

   $FORMATTED_CSR_KEY_PEM = "`"-----BEGIN PRIVATE KEY-----\n`"`n`"" + $($base64pkcs8PrivateKey -replace "[`r]*`n", "\n`"`n`"") + "\n`"`n`"-----END PRIVATE KEY-----\n`"";
   echo "static const char* csrPrivateKey =`n$FORMATTED_CSR_KEY_PEM;";

   $FORMATTED_CSR_PEM = "`"-----BEGIN CERTIFICATE REQUEST-----\n`"`n`"" + $($CSR_BASE64 -replace "(.{64})", "`$1\n`"`n`"") + "\n`"`n`"-----END CERTIFICATE REQUEST-----\n`"";
   echo "static const char* certificateSigningRequest =`n$FORMATTED_CSR_PEM;";
   ```

1. Configure and build the IoT SDK.

    ```sh
    # At the root of the repository
    cd build
    cmake -DCMAKE_BUILD_TYPE=Debug ..
    cmake --build . -j
    ```

## Run the sample

```sh
# At the root of the repository
cd build
./iothub_client/samples/iothub_client_sample_certificate_signing_request/iothub_client_sample_certificate_signing_request
```
